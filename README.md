# CustomDashboard_UI
A blue palette-themed dashboard customized for myself. 

Hello! 
With this project, I wanted to feature a recent project I had worked on since March 2026; I'm not gonna lie, the majority was learning some javascript. I was already familiar with CSS/HTML prior, due to a class I took in the Fall at my university. 

This project features a blue-themed widget-style dashboard that I designed for personal use; but for any regular viewer, it can also be a way to get to know me through interaction with the UI. For example, one of the more obvious choices of design was a blue color palette for the dashboard; my favorite color is blue, but I also like various shades of blue. 

I was inspired by the Notion and Google Calendar while making this; I loved the fluidity of the UI for both, and how easy it was to utilize both platforms. However, I wanted to make a more condensed verison, mixing both and creating my own, personal dashboard.
